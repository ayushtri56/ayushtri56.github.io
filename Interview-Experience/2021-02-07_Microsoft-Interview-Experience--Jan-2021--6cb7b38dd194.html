1. There are a total of n modules you have to take labelled from 0 to n - 1.

Some modules may have prerequisites, for example, if prerequisites[i] = [ai, bi] this means you must complete the module bi before the module ai.

Given the total number of modules numModules and a list of the prerequisite pairs, return the ordering of modules you should take to resolve the dependency of all modules.

If there are many valid answers, return any of them. If it is impossible to resolve all modules, return an empty array.

Approach: Topological Sort with BFS
